Los jugadores disponen del siguiente listado de controles para jugar e interactuar con la [[Interfaz de Usuario]].

- **Movimiento**:
	- **WASD** para desplazar al [[Influencer Foodie]].
	- **Mouse** para rotar la cámara.
	- **Ctrl Izq** o **C** para agacharse.
	- **Shift Izq** para correr.

- **Interacción**: con la tecla E los jugadores pueden realizar una de las siguientes acciones dependiendo del contexto:
	- Lootear el objeto seleccionado.
	- Sostener el objeto seleccionado.
	- Esconderse dentro del objeto seleccionado.

- **Interfaz de Usuario**:
	- **Esc** o **P** para abrir y cerrar el menú de pausa.
	- **Click Izq** para seleccionar elementos de la interfaz.

- **Chat de voz**:
	- **V** para activar y desactivarlo
